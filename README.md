# TURBO SPEED 3D Racer 🏎️

A fast-paced, browser-based 3D racing game built with React and Three.js. Navigate through a challenging track, master the art of drifting, and beat your best lap time!

## 🌟 Features

- **3D Graphics:** Fully rendered 3D environment using React Three Fiber.
- **Physics-Based Handling:** Realistic acceleration, friction, and steering mechanics.
- **Drift System:** Hold `Shift` to drift around tight corners for a sharper turning radius.
- **Nitro Boost:** Hit `Spacebar` for a sudden burst of speed to shave seconds off your lap time.
- **Lap Timer:** Tracks your current lap time and records your best lap.
- **Dynamic Camera:** Smooth follow-camera that tracks the car's movement and rotation.
- **Vibrant Environment:** Bright, colorful lighting with decorative scenery.

## 🎮 Controls

| Key | Action |
| :--- | :--- |
| **W** / **Up Arrow** | Accelerate |
| **S** / **Down Arrow** | Brake / Reverse |
| **A** / **Left Arrow** | Steer Left |
| **D** / **Right Arrow** | Steer Right |
| **Spacebar** | Drift |
| **Shift** | Nitro |

## 🛠️ Technologies Used

- [React](https://reactjs.org/) - UI Framework
- [Three.js](https://threejs.org/) - 3D Library
- [React Three Fiber](https://docs.pmnd.rs/react-three-fiber/getting-started/introduction) - React renderer for Three.js
- [React Three Drei](https://github.com/pmndrs/drei) - Useful helpers for R3F
- [Tailwind CSS](https://tailwindcss.com/) - Styling for the UI overlays
- [Vite](https://vitejs.dev/) - Build tool

## 🚀 Getting Started

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation
1. Clone the repository or download the source code.
2. Install the dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Open your browser and navigate to the local URL provided by Vite (usually `http://localhost:3000` or `http://localhost:5173`).

## 🏎️ Gameplay Tips
- **Master the Drift:** Use the `Shift` key on sharp corners to increase your steering radius without losing too much momentum.
- **Save your Nitro:** Use the `Spacebar` on long straightaways to maximize your top speed.
- **Find the Racing Line:** Cut corners smoothly to achieve the best possible lap time!
